## Sudoku Game - Android Application

A simple Sudoku game written in Kotlin, there are three levels, easy, medium, hard.

### Demo


### Features
- New game
- Exit 
- Delete cells
- Hint
- Reset
- Highlight the row, column and grid 3x3 in where the selected cell is
